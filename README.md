## 📋 Current Guides

### Paradox Junction — Black Ops 7 Zombies
> Season 2 Reloaded | Released March 11, 2026

A complete main quest Easter egg walkthrough for the Paradox Junction map. Covers all 26 steps from Pack-a-Punch through the Dark Heart boss fight — solo and co-op friendly.

**Features:**
- Collapsible step-by-step sections
- Progress tracker with step counter
- Sticky navigation with jump links
- Mark Complete checklist for each step
- Loadout and gobble gum recommendations
- Boss fight phase breakdowns

## ♿ Accessibility

All guides in this repository are built to **WCAG 2.1 AA** compliance standards, including:

- Minimum 4.5:1 color contrast ratios on all text (body text exceeds 7:1)
- Minimum 16px body text size throughout
- Full keyboard navigation support
- Screen reader compatible — semantic HTML, ARIA labels, `aria-expanded`, live regions
- Skip-to-content link
- 44×44px minimum touch targets on all interactive elements
- `prefers-reduced-motion` support
- Color is never used as the sole means of conveying information
- All external links identified as opening in a new tab

## 🚀 Deploying with GitHub Pages

1. Go to your repository **Settings**
2. Select **Pages** from the left sidebar
3. Under **Source**, select **Deploy from a branch**
4. Set branch to `main` and folder to `/ (root)`
5. Click **Save**

Your guide will be live at:
`https://belovedcoachp.github.io/ZombiesTips/`
